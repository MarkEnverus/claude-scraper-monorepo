# MISO Coordinated Transaction Scheduling (CTS) Scraper

Collects MISO-PJM interface forecasts from MISO's public CTS API.

## Data Source
- **URL**: https://public-api.misoenergy.org/api/CoordinatedTransactionScheduling
- **Format**: JSON
- **Update**: Every 15 minutes
- **Historical**: Yes - date parameter supports past dates
- **Authentication**: None

## What is Coordinated Transaction Scheduling?

Coordinated Transaction Scheduling (CTS) provides forecasted Locational Marginal Prices (LMPs) and transaction volumes for the MISO-PJM interface. This data is essential for cross-market arbitrage opportunities and inter-regional trading strategies.

## Data Format
- **Structure**: JSON with data array and metadata object
- **Intervals**: 5-minute market intervals (288 per day)
- **Nodes**: Hundreds of MISO-PJM interface nodes
- **LMP Components**: Energy, congestion, losses
- **Transaction Data**: Volume and direction between MISO and PJM
