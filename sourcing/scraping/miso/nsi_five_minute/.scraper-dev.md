# MISO NSI Five-Minute Scraper

This scraper collects Net Scheduled Interchange (NSI) data from MISO's public API at 5-minute intervals.

## Data Source
- **ISO**: MISO (Midwest ISO)
- **Endpoint**: https://public-api.misoenergy.org/api/Interchange/GetNsi/FiveMinute
- **Update Frequency**: Every 5 minutes
- **Data Format**: JSON
- **Authentication**: None required (public API)

## What is NSI?
Net Scheduled Interchange (NSI) represents the net amount of power scheduled to flow into or out of MISO's control area. It's a key metric for understanding real-time interchange operations between MISO and neighboring ISOs/utilities.

## Data Characteristics
- **Historical Data**: No - API returns only current/latest data
- **Real-time**: Yes - updates every 5 minutes
- **Rate Limits**: Reasonable use (no strict limits documented)
- **Pagination**: Not applicable - single endpoint returns current value
