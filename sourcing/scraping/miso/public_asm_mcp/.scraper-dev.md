# MISO Public Ancillary Services Market Clearing Price (MCP) Scraper

Collects real-time and day-ahead reserve market clearing prices from MISO's public API.

## Data Source
- **URL**: https://public-api.misoenergy.org/api/MarketPricing/GetAncillaryServicesMcp
- **Format**: JSON
- **Update**: Every 5 minutes (real-time + day-ahead snapshot)
- **Historical**: Limited (via datetime parameter)
- **Authentication**: None (public API)

## What is Ancillary Services MCP?

Market Clearing Price (MCP) for Ancillary Services represents the cost of procuring reserve capacity to maintain grid reliability. This data shows the market-clearing prices for various reserve products across MISO's 8 reserve zones.

## Reserve Products

1. **Regulation**: Fast frequency response (seconds)
2. **Spin**: Spinning reserves (10 minutes)
3. **Supplemental**: Non-spinning reserves (10 minutes)
4. **STR**: Short-term reserves
5. **Ramp-up**: Increase generation capability
6. **Ramp-down**: Decrease generation capability

## Reserve Zones

- Zone 1 through Zone 8 (MISO's reserve zones)

## Market Types

- **real-time**: Current operating hour prices
- **day-ahead**: Next-day forecasted prices

## Data Format

Returns JSON with:
- **data**: Array of MCP records (datetime, product, zone, mcp, marketType)
- **metadata**: totalRecords, updateTimestamp, dataSource

## Query Parameters (Optional)

- **product**: Filter by reserve product name
- **zone**: Filter by reserve zone (Zone 1-8)
- **datetime**: Filter by specific ISO 8601 datetime

## Collection Strategy

Since this is real-time data updated every 5 minutes, the scraper collects snapshots. Optional filters can reduce data volume by product, zone, or datetime.

Expected volume without filters: 96 records per collection (6 products × 8 zones × 2 market types).
